## 용수철
용수처릥 변형한 정도에 따라 힘의 크기와 방향이 계속 변한다.
### 훅의 법칙 = 용수철 힘
$$F_{spr}=-K_s(l-l_0)$$
용수철이 변형된 거리에 비례하여 반대 방향에 힘이 작용한다.  
$k_s$ 용수철 상수: 용수철의 뻣뻣한 정도이다.  
#### 3차원 힘의 크기
$$|\vec{F}_{spr}| = k_s(|\vec{l}| - l_0)$$
#### 힘의 방향
$$\hat{F}_{spr} = -\hat{l}$$
#### 다 붙이면
$$\vec{F}_{spr} = - k_s(|\vec{l}| - l_0)\hat{l}$$
